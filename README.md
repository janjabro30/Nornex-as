# Nornex IT Services Platform

A professional, responsive website for Nornex - an IT services company offering Managed IT, Cybersecurity, and Web Development solutions.

## Overview

This website serves as the online presence for Nornex, showcasing:
- **Managed IT Services** - 24/7 monitoring, help desk support, cloud management
- **Cybersecurity Solutions** - Threat detection, compliance management, security training
- **Web Development** - Custom websites, e-commerce, web applications

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required

For local development with live reload, you can use any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve

# Using PHP
php -S localhost:8000
```
